└──────────────────────────────────────────────────────────────────────────────┘

  ❌ Server window closes immediately
     → Check: Java is installed (java -version)
     → Check: MySQL running (mysqld process)
     → Check: lib\mysql-connector-j-8.3.0.jar exists

  ❌ Admin/Customer window shows "Connection refused"
     → Wait: Server needs 3 seconds to start RMI registry
     → Auto-retries every 5 seconds - just wait
     → Check: Server window still open and running

  ❌ Database connection error
     → MySQL credentials: root (no password)
     → Database: drinks_system (case-sensitive)
     → Port: 3306
     → URL: jdbc:mysql://localhost:3306/drinks_system

  ❌ Port 1099 already in use
     → Kill other Java processes
     → Or restart computer
     → Or change port in ServerMain.java

  ❌ Order fails with "Out of stock"
     → Current stock: 100 units per item per branch
     → Reduce order quantity
     → Or check Low Stock Alerts for actual inventory
