COMMUNICATION FLOW:
1. Client sends RMI request to Server
2. Server unmarshals request, executes business logic
3. Server executes JDBC queries to Database
4. Database returns results (SQL ResultSet)
5. Server marshals results back to Client
6. Client receives RMI response, updates UI

CONCURRENCY MODEL:
• Thread-Safe Stock Updates: ReentrantLock (java.util.concurrent.locks)
• Order Processing: Atomic transactions with rollback
• Multiple Clients: RMI registry supports unlimited concurrent connections
• Database: Connection pooling prevents connection exhaustion

════════════════════════════════════════════════════════════════════════════════
                           TROUBLESHOOTING GUIDE
════════════════════════════════════════════════════════════════════════════════

ISSUE: "MySQL Driver not found"
  SOLUTION: Verify lib\mysql-connector-j-8.3.0.jar exists (2.4 MB)
  FIX: Redownload by running Java command above, classpath must include:
       -cp "out;lib\mysql-connector-j-8.3.0.jar"

ISSUE: "Connection refused: Cannot bind to port 1099"
  SOLUTION: Another process using port 1099
  FIX: Kill other Java processes or change RMI port in ServerMain.java

ISSUE: "java.sql.SQLException: User does not exist"
  SOLUTION: XAMPP MySQL doesn't have given credentials
  FIX: Use root user (no password) as configured in DatabaseConnection.java

ISSUE: "Cannot find database 'drinks_system'"
  SOLUTION: Database hasn't been created
  FIX: Run database\setup.sql manually:
       mysql -u root < database\setup.sql

ISSUE: Client shows "✗ Error - Connection refused"
  SOLUTION: Server not running on localhost:1099
  FIX: Ensure ServerMain window shows "✓ SERVER RUNNING SUCCESSFULLY"
       Client will auto-retry every 5 seconds

ISSUE: "Exception in thread AWT-EventQueue" in AdminClient
  SOLUTION: Network error during report fetch
  FIX: Check server is still running, click report button again to retry

ISSUE: Order fails with "Insufficient stock"
  SOLUTION: Not enough inventory for requested quantity
  FIX: Check Low Stock Alerts report, reduce order quantity, or restock
